# AMOGUS
Esolang Based on Amogus Meme
# Tutorial
**Your program should consist of only two lines.**
Here is Commands:

```AMOGUS```(Only For First Line) - Print Next Line;

```sus```(For First Line),```SUS```(For Second Line) - Pause, Exit
# Examples
```
AMOGUS
Hello, World!
```
Prints "Hello, World!" To The Screen
